# Project Repositories & Codenames

ウィーアワーズ・インテリジェンス社のプロジェクトは、機能的な役割だけでなく、それぞれの性格や相互関係を表すコードネーム（Metaphor）を持っています。
開発チームは、これらの物語性を大切にしながら設計・実装を行っています。

## 1. Project: DEAIAS (The Engine)
基幹AIエンジン。脳構造と艦隊行動をメタファーとしています。

| Repository Name | Codename | Role & Metaphor |
| :--- | :--- | :--- |
| **`deaias-core`** | **Core** | **中核・合意形成**<br>評議エンジンの中枢。システムの「脳」として機能し、全ての意思決定とルーティングを司ります。 |
| **`deaias-memex`** | **Memex** | **拡張記憶**<br>Vannevar Bushの提唱した「Memex」に由来。商用モデルが持たない知識を補完し、自己蒸留によって成長し続ける「記憶」の保管場所です。 |
| **`deaias-fleet`** | **Fleet** | **思考艦隊**<br>並列処理ワーカー群。セマンティック・チャンキングとMap-Reduce処理を行う、分散した「思考の船団」です。 |

## 2. Project: ZAPETTE (The Media)
新世代デジタルマガジン。「Zapping」と「Gazette」を掛け合わせたブランド名に基づいています。

| Repository Name | Codename | Role & Metaphor |
| :--- | :--- | :--- |
| **`zapette-zapper`** | **Zapper** | **チャンネル切替機**<br>ユーザー体験（Frontend）の中核。「チャンネルを変えるように」軽快にコンテンツを切り替え、没入させるアクションを体現しています。 |
| **`zapette-gazella`** | **Gazella** | **ガゼル**<br>優雅さと速さ。「Gazette（新聞）」の響きを持ちつつ、情報のデリバリーを俊敏に行うバックエンドAPIのスピード感を表現しています。 |

## 3. Project: Sidekick (The Partner)
パーソナルエージェント。「ヒーロー（ユーザー）を支える相棒と装備」という関係性を構築します。

| Repository Name | Codename | Role & Metaphor |
| :--- | :--- | :--- |
| **`sidekick-jetpack`** | **Jetpack** | **推進装備**<br>映画『Kick-Ass』のクライマックスで主人公が使用したガトリング付きジェットパック。最強の相棒（Mindy）と共に空を飛び、困難（敵地）へ到達するための「最強のツール/UI」です。 |
| **`sidekick-mindy`** | **Mindy** | **最強の幼女/相棒**<br>『Kick-Ass』のHit-Girl（Mindy Macready）。未熟なヒーロー（ユーザー）を圧倒的な実力で影から支え、導くバックエンド・ロジックの頼もしさを表現しています。 |
