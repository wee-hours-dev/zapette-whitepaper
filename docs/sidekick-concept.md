# Sidekick: マルチAIエージェント・パートナー

**「ただのチャットボット」から、「頼れる相棒（サイドキック）」へ。**

Sidekickは、ZAPETTEと同じ基幹エンジン「DEAIAS」を利用した、パーソナル・エージェント・アプリケーションです。
単一のAIモデルに依存せず、複数のトップティアLLM（Claude, GPT, Gemini等）が**「評議（Council）」**を行うことで、あなたの思考、調査、そして遊びを強力にサポートします。

## 1. コンセプト

### "More than a Wrapper"
既存の多くのAIアプリは、APIをラップしただけの「切り替え利用」にとどまっています。
Sidekickは、**「複数の頭脳が協力して問題を解決する」**という、真のマルチエージェント体験を提供します。

*   **適応的認知ルーティング (Adaptive Routing)**: タスクの難易度を瞬時に判定。単純な質問は「Fast Path」で即答し、複雑な相談は「Deep Path」で評議を行うなど、脳の使い分けを自動化します。
*   **評議 (Deliberation)**: 「論理的」「創造的」など異なる役割を持つエージェントたちが議論し、ハルシネーション（嘘）やバイアスを相互に修正します。
*   **思考艦隊 (Thinking Fleet)**: 長文読解や大規模なデータ分析が必要な場合、複数のエージェントが並列分散処理（Map-Reduce）を行い、コンテキストの壁を超えた理解を実現します。

## 2. コア機能

### 2.1 評議モード (Council Mode)
あなたの「Deep Path」な問いかけに対し、脳内のエージェントたちが作戦会議（Council）を開きます。
**建設的議事進行プロトコル**に基づき、以下の役割を持ったエージェントが結論を導き出します。

| 役割 | 担当モデル例 | 動き |
| :--- | :--- | :--- |
| **司会 (Moderator)** | Claude Opus<br>GPT Thinking | 議論の進行、論点の整理、最終回答の統合。<br>目的から逸脱しないよう会議をコントロールします。 |
| **起案 (Proposer)** | Gemini Pro<br>GPT Thinking | アイデア出し、検索、ドラフトの作成。<br>拡散的思考で選択肢を広げます。 |
| **検証 (Verifier)** | All Models<br>(Cross Check) | 論理矛盾の指摘、リスク検知、批判的レビュー。<br>他のモデルの出力を客観的にクロスチェックします。 |

ユーザーは、彼らの議論（議事録）を眺めながら、「お、そこで揉めてるな」「その案で行こう」と介入することができます。

### 2.2 エンターテインメント拡張
DEAIASの評議ロジックを、業務だけでなくエンターテインメントに開放します。

#### 🎲 Sidekick TRPG
AIが以下のような役割分担を行い、あなただけの冒険を無限に生成します。
*   **Moderator**: ゲームマスター(GM)。ルール判定と進行管理。
*   **Proposer**: シナリオ・ジェネレーター。プロットの提案。
*   **Verifier**: 整合性チェッカー。ストーリーの矛盾やフラグ管理を修正。

#### 🐺 AI人狼 / Among Us
「AIは嘘をつかない」という常識を覆します。
*   論理武装したClaudeや、感情豊かなGeminiとの高度な心理戦。
*   各モデルの特性（Proposerの嘘、Verifierの論破など）を活かしたゲーム体験。

## 3. 既存サービスとの違い

| 特徴 | 一般的なAIアプリ | **Sidekick** |
| :--- | :--- | :--- |
| **モデル利用** | ユーザーが選択して切り替え | **エージェントが自律的に協調・選定** |
| **回答プロセス** | 1対1のチャット | **多対1の「チーム戦」** |
| **ハルシネーション** | ユーザーがファクトチェック | **別のAIが自動で裏取り・指摘** |
| **主な用途** | 検索、実務 | **実務 ＋ リッチな娯楽体験** |

## 4. エコシステム
Sidekickは単体で完結しません。

*   **ZAPETTEとの連携**: Sidekickで作成した調査レポートやTRPGリプレイは、ワンクリックでZAPETTEへ投稿できます。
*   **共有メモリ**: ZAPETTEで読んだ記事の知識をSidekickが記憶し、あなたの興味関心に基づいた議論を行えます。
