# ZAPETTE: 新世代デジタル・マガジン

### ZAPETTE（ザペット ＝ Zapping + Gazette）
### 報道から娯楽まで。多様な知性と感性が交差する、新しいパブリッシング・エコシステム

## 1. ビジョン

マスメディアの構造的バイアスや、SNSのアテンション・エコノミー（バズ至上主義）から解放された、自由で開かれた「雑誌」の再定義。
専門的なジャーナリズムから、クリエイティブなエンターテインメントまで、あらゆる「質の高い表現」が正当な対価を得られるデジタル民主主義を実装します。

## 2. 解決する課題

* **画一化された情報空間**: 巨大資本やスポンサーの意向に左右される報道と、アルゴリズムに消費される刹那的な娯楽。
* **「煽り」と「バズ」の弊害**: PV数だけが収益を生む構造により、手間をかけた調査報道や、余韻のある良質な創作物が埋没する現状。
* **購読体験の断片化**: 個別課金による情報の囲い込み（サイロ化）が、読者の好奇心を制限し、経済的負担を増大させている課題。

## 3. 主要な特徴

* **記者・クリエイターによる直接投稿**: 運営が記事・コンテンツを発注・収集するのではなく、記者やクリエイターが自らの責任で自由にコンテンツを投稿し、ユーザーに届ける仕組みです。SNSのような自由さと、雑誌のような深みが共存する空間を目指します。
* **十字スクロール・インターフェース**:
  * **Y軸（目次）**: 多彩な見出しを zapping し、偶然の出会いを楽しむ。
  * **X軸（ページ）**: 気になったコンテンツの「奥」へ入り、深く没入する。 画面遷移を排した、デジタルならではの「ページをめくる」体験。
* **共有プール型サブスクリプション**: 読者はマガジン全体に課金。特定の誰かではなく「良い仕事」をしたクリエイターたちへ、読者の支持（読了・評価）に基づいてプール金から分配されます。
* **AI執筆・創作支援**: 投稿者向けに、AIエージェント（DEAIAS）による推論・校閲リソースを提供。ファクトチェックの補助や論理構成の壁打ちを支援し、個人ジャーナリストのリサーチコストを劇的に低減します。
* **クリエイターの盾（リーガル・校閲基金）**: 収益プールの10%を積み立て。事実確認のサポートや、不当な法的リスクから、個人の表現活動を組織的に守ります。

## 4. フェアな評価と分配（クオリティ・スコアリング）

ZAPETTEでは、瞬間的なアクセス数（PV）の多寡ではなく、コンテンツ의「価値」を多角的に評価し、分配額を決定します。

### 高く評価される要素（報酬が増える）
* **最後まで読ませる力**: 読了率を最重視します。読者の心を掴み、最後まで誠実に消費されたコンテンツほど高く評価されます。
* **情報の信頼性と整合性**: エビデンスに基づいた記述や、AIエージェント（DEAIAS）による監査をクリアした論理的整合性の高い内容は加点されます。
* **専門家・コミュニティの支持**: 実績のある他のクリエイター（査読者）からの高い支持は、分配額に強力なポジティブ・バイアスを付与します。

### 評価されない・制限される要素（報酬が抑えられる）
* **煽り・クリックベイト**: タイトル詐欺や中身の薄い「釣り」記事は、読了率の低さによって自然と評価が制限されます。
* **組織的な不正操作**: Botやクリックファームによる不自然なアクセスの急増は、統計的フィルタリングによって無効化されます。
* **根拠のない誹謗中傷**: 信頼性スコアが著しく低い内容は、分配対象から除外される冷却期間が適用されます。

## 5. 基幹技術：DEAIAS (DEliberating AI Agents System = 評議・協調AIエージェント・システム）

ウィーアワーズ・インテリジェンス社の基幹技術である、マルチモデルAIエージェント・エンジン。複数のAIモデルが協調・評議を行うことで、単一モデルを凌駕する客観性と論理的洞察を実現します。

### DEAIAS エコシステム
DEAIASは以下のサービスを通じて、社会の知性をアップデートします。

* **ZAPETTE**: 本デジタルマガジン。DEAIASを「信頼性の担保（AI監査）」と「分配の適正化（不正検知）」に活用し、質の高い表現を支援します。
* **Sidekick**: パーソナルAIエージェントアプリ。DEAIASの評議ロジックを個人の思考、調査、意思決定の壁打ちに開放します。
